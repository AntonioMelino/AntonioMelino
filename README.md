## **🚀 Proyectos Destacados**

### **[💼 Portfolio Personal](https://portfolio-antonio-melino.vercel.app/)**
Sitio web profesional desarrollado con **React** y desplegado en **Vercel**. Presenta mi experiencia, habilidades y proyectos de manera moderna y responsive.

### **[🎮 Gamer Zone App](https://gamer-zone-app.vercel.app/)**
Tienda e-commerce gamer completa construida con **React** y **Firebase (Firestore)**. Incluye:
- ✅ Autenticación de usuarios con JWT
- ✅ Carrito de compras en tiempo real
- ✅ Filtros por categoría
- ✅ Panel de administración
- ✅ Diseño responsive con **Material-UI**

### **[🏠 Techos Nass](https://techos-nass.vercel.app/)**
Página web corporativa moderna para empresa especializada en techos. Tecnologías:
- **TypeScript** y **Next.js**
- Sistema de reseñas de Google integrado
- Mapa interactivo con ubicación del negocio
- Formulario de contacto con WhatsApp API
- Diseño completamente responsive
- Desplegado en **Vercel**

### **[🏗️ BSM Techos y Zinguería](https://bsm-techos-y-zingueria.vercel.app/)**
Página web corporativa para empresa de techos y zinguería. Características:
- **TypeScript** y **Next.js**
- Formulario de contacto con integración a WhatsApp
- Diseño responsive y moderno
- Despliegue optimizado en **Vercel**

### **[🔍 Verificador de Fotos para Auditorías](https://verificador-de-fotos.web.app/)**
Herramienta web profesional para auditorías visuales. Stack:
- **HTML, CSS y JavaScript** puro
- Carga y procesamiento de imágenes
- Identificación de desvíos en fotos
- Conversión PDF a JPG
- Exportación a Excel
- Librerías: **SweetAlert2** y **JSZip**
